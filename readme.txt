This plugin will help you to understand what your visitors like to do on your website.

== Description ==
This plugin will tell you the most valuable information about your site visitors:
<ul>
	<li>Registered users:
		<ul>
			<li>When your registered users are logging</li>
			<li>Where they go when they are logged</li>
			<li>Where are they from</li>
		</ul>
	</li>
	<li>Unregistered users:
		<ul>
			<li>Know where your unknown visitors are going on your site</li>
			<li>View from where on the map did they accessed your website</li>
			<li>Track information about those potential users and manage to convert them into real users</li>
		</ul>
	</li>
</ul>

This information can be found under <strong>Settings > <u>WPSK Statistics</u></strong>

<strong>Note:</strong>
This plugin uses the <strong><a href="http://ipinfo.io/" target="_blank">ipinfo.io</a></strong> API for converting IP address to real world address information, which allows you to know your visitors better.

You can join the plugin development on GitHub - <a href="https://github.com/Gero0Nikolov/WP-Spykey" target="_blank">https://github.com/Gero0Nikolov/WP-Spykey</a>
